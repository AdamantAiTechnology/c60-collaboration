# SOULC60 — Landing Page Prototype

Премиальный лэндинг для коллаборации бренда медитаций и гипнотической деструкции родовых травм с фуллереном C60.

## О проекте

**SOULC60** — концептуальная точка пересечения трёх трансформационных направлений:
- Глубинные медитативные практики
- Гипнотическая деструкция родовых и личных травм (авторский метод)
- Молекула фуллерен C60 (антиоксидант нового поколения)

**Целевая аудитория лэндинга:** партнёры и владельцы C60-брендов — для привлечения их к коллаборации без необходимости иметь кейсы или портфолио.

## Визуальные элементы

- **Canvas анимация C60-молекулы** в Hero — вращающийся граф молекулярных связей (золотые узлы и рёбра)
- **Canvas анимация молекулы** в секции Experience — 3D-вращение икосаэдра
- **Particle field** на фоне CTA-секции
- **Venn-диаграмма** синергии (CSS)
- **Hover-эффекты** на карточках с радиальным градиентом
- **Scroll reveal** анимации для всех карточек
- **Counter анимация** для числовых значений
- **Burger-меню** для мобильных устройств

---

## Технологии

- **HTML5 / CSS3 / Vanilla JS** — без фреймворков
- **Google Fonts:** Cormorant Garamond (serif, для заголовков) + Inter (sans, для текста)
- **Font Awesome 6** — иконки
- **Canvas API** — анимации молекул и частиц
- **IntersectionObserver** — scroll reveal и counter trigger
